## Basic setup steps 
<!-- Install virtual env -->
    python3 -m venv venv  

<!-- activate environment -->
    source venv/bin/activate (cmd is a bit different on Windows) 

<!-- Install pip if needed -->

<!-- Install Flask -->
    pip install flask

### Project commands
<!-- Initialize DB -->
    flask --app flaskr init-db

<!-- Run the app local -->
    flask --app flaskr run --debug
